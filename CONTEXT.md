# Soul Garden Sound

## What This Is
Wellness/Reiki practice website for Lindsay. CMS-driven so she can edit content without developer help.

## Stack
React 19 + TypeScript + Vite + Tailwind v3 + Decap CMS (GitHub OAuth) + Vercel.

## Status
**Live** — soul-garden-sound.vercel.app. Stable, minimal maintenance.

## Key Files
- `src/content/` — Markdown content files (services, events, testimonials)
- CMS config for Decap CMS with GitHub OAuth

## What's Built
- Modern responsive design
- Decap CMS integration for non-technical content editing
- Content types: Services, Events, Testimonials (stored as Markdown)
- Auto-deploys via Vercel from GitHub

## What's Next
- No urgent tasks. Stable and live.
